list_records(self, from_date=None, until_date=None, oai_set=None, list_size=None, start_number=None, **kwargs)

source code 

List the records which match the current set of parameters.

Note that this function ignores the until_date and the oai_set, because:

  • until_date - there is no equivalent operation in JPER, so it is not possible to give an accurate count of a time-boxed request

  • oai_set - This interface does not implement sets

Parameters:
  • from_date - The date to request notifications from
  • until_date - DO NOT USE
  • oai_set - DO NOT USE
  • list_size - number of results per page (subject to limitation by JPER)
  • start_number - record number to start this page with
  • kwargs - additional kewyword arguments to pass to JPER.list_notifications
Returns:
a tuple of the total count, and the list of notifications on this page: (count, this_page)
