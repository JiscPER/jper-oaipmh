container_exists(self, path)

source code 

Does the url path provided refer to a notification that already exists?

Note that this queries the JPER API and caches a copy of the notification

Parameters:
  • path - url path (e.g. the notification id)
Returns:
True if the notification exists, False if not

service_document(self, path=None)

source code 

Construct the Service Document for JPER. This takes the set of collections that are in the store, and places them in an Atom Service document as the individual entries

This will provide two collections for deposit: one for validation requests and the other for create requests.

Parameters:
  • path - url path sent to the server (for supporting sub-service documents, which we don't in this implementation)
Returns:
serialised service document
